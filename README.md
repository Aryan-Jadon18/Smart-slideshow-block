# Smart Slideshow — WordPress Block

[Live Demo](https://smart-slideshow-rtcamp.vercel.app/) • Demo deployed on Vercel

A lightweight, accessible, and responsive Gutenberg block that creates automatic sliding/carousel layouts you can drop into any post or page.

Badges: ![License](https://img.shields.io/badge/license-GPL--2.0-brightgreen)

---

## Quick demo

Open the Live Demo linked above to see the block in action. The block supports autoplay, custom slide duration, pause-on-hover, and simple content slides (images, headings, text, and buttons).

## Features

- Gutenberg-native block, no shortcodes required
- Autoplay with configurable speed and easing
- Pause on hover / pause on focus (accessibility-friendly)
- Responsive behavior and touch/swipe support
- Simple UI controls for number of slides, dots, and arrows
- Work well with images, rich text, and nested blocks
